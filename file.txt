set showmode:查看当前模式
h j k l ==左 下 上 右
dd:删除一整行
u：撤销上一次操作
x:删除光标字符
J:删除换行符
CTRL+R：撤销前一个命令
U：行撤销
o:在光标下方新建空白行并开始插入
O:在光标上方新建空白行并开始插入
命令前方加数字表示执行几次
ZZ:保存并退出（不会产生备份并且修改立即生效，不能恢复）
:q!:强制退出
:e!:继续修改
w:向前跳动到下一个词首
b:向后跳动到上一个词首
e:下一个单词的词末
ge:上一个单词的词末
$:光标移动到行尾
^:光标移动到第一个非空字符
0:移动到第一个字符
f:向前寻找字符
F:向后寻找字符
%:括号匹配
数字+G:移动到指定行
G:移动到文件末尾
gg:移动到文件首部
数字+%:文件大概位置
H/M/L:文件的头部/中间/尾部
CTRL+G:确定当前行在文件中的位置
set number/nonumber:在每一行前面加上/关闭行号
CTRL+U/D:窗口向上/下移动半屏，文字向下/上移动半屏
CTRL+E/Y:上/下滚一行
CTRL+F/B:正向/反向滚动一行
zz/zt/zb:光标移动到屏幕中间/顶部/底部
/string:查找单词 n即查找下一个 N查找上一个
?string:反向查找 n反向查找，N正向查找
set ignorecase:忽略大小写 set noignorecase:再次区分大小写
*:向前查找光标所在单词 #:向后查找 n/N的用法同/？
\>:词尾匹配 \<:词首匹配 *和#使用词首和词尾匹配，若需要部分匹配，使用g* g#
set hlsearch:高亮匹配 set nohlsearch:关闭高亮
set incsearch:在输入过程中显示匹配点
set wrapscan:默认是开启，查找时遇到文件开头或者结尾停止查找，如需关闭加no
'':跳转到原来的位置 跳转包括/和?，不包括w e h j k l
CTRL+0:跳转到较老的地方 CTRL+I:跳转到较新的地方
jumps:显示跳转列表
ma:用a标记当前位置 'a:跳转到a标记的位置 marks:获得标记列表
d+motion:删除光标当前到移动目标位置之间
c+motin:与d类似，但会进入插入模式 一个例外 cw不会删除单词前的空格，只删除到单词尾 历史遗留习惯
cc:修改一整行 c$:修改到行尾
r+字符:修改光标位置的字符 无需退出插入模式
.:重复一个修改，除了u和CTRL+R
v:进入可视模式 V:按行选择 CTRL+V:按列选择
可视模式下o跳到选中文字另一端，按列选择时只会移动到对角，O移动到同一行另一个角
p:粘贴删除的文本 xp:交换两个字符
y:拷贝文本 yy/Y:拷贝一整行 y$:拷贝到行尾 "*y "*p:跨软件复制和粘贴
daw:删除一个单词 is:内含句子 as:外含句子，包括空白符
R:进入替换模式 退格撤销替换
~:改变光标处大小写
A/I:移动到第一个非空字符/行尾并启动插入
