set showmode:查看当前模式
h j k l ==左 下 上 右
dd:删除一整行
u：撤销上一次操作
x:删除光标字符
J:删除换行符
CTRL+R：撤销前一个命令
U：行撤销
o:在光标下方新建空白行并开始插入
O:在光标上方新建空白行并开始插入
命令前方加数字表示执行几次
ZZ:保存并退出（不会产生备份并且修改立即生效，不能恢复）
:q!:强制退出
:e!:继续修改
w:向前跳动到下一个词首
b:向后跳动到上一个词首
e:下一个单词的词末
ge:上一个单词的词末
$:光标移动到行尾
^:光标移动到第一个非空字符
0:移动到第一个字符
f:向前寻找字符
F:向后寻找字符
%:括号匹配
数字+G:移动到指定行
G:移动到文件末尾
gg:移动到文件首部
数字+%:文件大概位置
H/M/L:文件的头部/中间/尾部
CTRL+G:确定当前行在文件中的位置
set number/nonumber:在每一行前面加上/关闭行号
CTRL+U/D:窗口向上/下移动半屏，文字向下/上移动半屏
CTRL+E/Y:上/下滚一行
CTRL+F/B:正向/反向滚动一行
zz/zt/zb:光标移动到屏幕中间/顶部/底部
